# fis3-vue-template

## 简介
这是一个使用vue-cli来初始化项目的模板


## 如何使用

### 全局安装vue-cli,fis3

```sh
npm install -g vue-cli fis3
```

### 初始化项目
```sh
vue init sqliang/fis3-vue-template your-project
```

### 打开项目目录
```sh
cd your-project
```
### 安装依赖
```sh
npm install
```
### 运行项目

1. 打开fis3本地服务
```sh
fis3 server start
```

2. dev环境下部署运行项目(http://localhost:8080/{your-project-name}/index.html)
```
npm run dev
```
3. build 项目
```
npm run build
```
